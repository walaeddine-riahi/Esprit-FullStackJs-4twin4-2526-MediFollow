STRUCTURE DES DOCUMENTS:

Chaque document couvre:

📍 MODULE PATIENT
├── Chatbot Assistant (Azure OpenAI / GPT-4)
│   ├── API: POST /api/patient/chatbot
│   ├── Modèle: GPT-3.5-turbo
│   └── Fonctionnalités: Conversation, conseils santé

📍 MODULE VITAL SIGNS
├── Vital Parser (Groq / Llama 3.3)
│   ├── Service: lib/ai/vitalParser.ts
│   ├── Extraction: BP, HR, Temp, SpO2, Weight
│   └── Validation: Plages automatiques
├── AI Status Classification (Azure OpenAI / GPT-4o)
│   ├── Service: lib/services/vitals-ai-status.service.ts
│   ├── Output: Sévérité, insights, recommandations
│   └── Seuils: EXCELLENT → CRITICAL
└── Risk Analysis (Groq / Llama 3.3)
    ├── Service: lib/ai/riskAnalysis.ts
    ├── Score: 0-100 (Low → Critical)
    └── Output: Risque + tendances + action items

📍 MODULE INFIRMIÈRE
├── Report Generation (Groq / Llama 3.3)
│   ├── Service: lib/ai/reportGeneration.ts
│   ├── Max Tokens: 2000
│   └── Output: Rapport markdown formaté
└── Quick Summary (Groq / Llama 3.3)
    ├── Max Tokens: 200
    └── Output: Résumé 2-3 phrases

📍 MODULE ADMIN
└── Admin Copilot (Azure OpenAI / GPT-4)
    ├── Service: lib/ai/admin-intelligence.ts
    ├── Mode: JSON
    └── Output: Next best actions + confiance

📍 MODULE AUTHENTIFICATION
└── Face Recognition (TensorFlow.js / Face-API)
    ├── Component: FaceLoginModal.tsx
    ├── Modèles: tinyFaceDetector, faceLandmarkNet, faceRecognitionNet
    └── Features: Real-time detection, anti-spoofing

CONFIGURATION RAPIDE:

1. **Groq API**
   - Key: GROQ_API_KEY=gsk_...
   - Modèles: llama-3.1, llama-3.3, mixtral
   - Promo: Gratuit avec rate limit
   - URL: https://console.groq.com

2. **Azure OpenAI**
   - Key: AZURE_OPENAI_API_KEY=...
   - Endpoint: AZURE_OPENAI_ENDPOINT=https://xxx.openai.azure.com/
   - Deployment: AZURE_OPENAI_DEPLOYMENT=gpt-4
   - Version: AZURE_OPENAI_API_VERSION=2024-02-15-preview

3. **Santé Connect OAuth**
   - Client ID: SANTE_CONNECT_CLIENT_ID=...
   - Client Secret: SANTE_CONNECT_CLIENT_SECRET=...
   - Redirect: SANTE_CONNECT_REDIRECT_URI=...
