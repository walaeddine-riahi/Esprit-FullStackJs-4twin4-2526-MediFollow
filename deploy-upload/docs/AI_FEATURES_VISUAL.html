❤️ Module Constantes Vitales

1. Vital Signs Parser

Service: lib/ai/vitalParser.ts

Description: Extraction intelligente des constantes vitales à partir de transcriptions vocales

API Utilisée

Groq API - Llama 3.3 70B

Paramètres Extraits

  • Tension artérielle (systolique/diastolique)
  • Fréquence cardiaque
  • Température
  • Saturation en oxygène (SpO2)
  • Poids
  • Notes additionnelles

Validation Automatique

Vérification des plages normales :

  • Systolic BP: 50-250 mmHg
  • Diastolic BP: 30-150 mmHg
  • Heart Rate: 30-200 bpm
  • Temperature: 35-42°C
  • SpO2: 70-100%

2. AI Vital Status Classification

Service: lib/services/vitals-ai-status.service.ts

Description: Classification contextuelle intelligente au-delà des simples seuils

API Utilisée

Azure OpenAI - GPT-4o

Résultats Fournis

  • Sévérité (EXCELLENT → CRITICAL)
  • Classification clinique
  • Insights détaillés
  • Recommandations d'action
  • Facteurs de risque identifiés
  • Corrélations entre vitals

3. Risk Analysis Engine

Service: lib/ai/riskAnalysis.ts

Description: Analyse du risque patient basée sur historique et tendances

API Utilisée

Groq API - Llama 3.3 70B

Score de Risque

0-30 🟢 Risque Faible
31-50 🟡 Risque Modéré
51-75 🟠 Risque Élevé
76-90 🔴 Risque Très Élevé
91-100 ⚫ Risque Critique

🔐 Module Authentification

Face Recognition Login

Component: components/FaceLoginModal.tsx

Description: Authentification sécurisée et rapide par reconnaissance faciale

Technologie

@vladmandic/face-api + TensorFlow.js

Fonctionnalités

  • Détection faciale en temps réel
  • Mesures anti-spoofing
  • Quality validation
  • Fallback sur password
  • Support 2FA

Modèles Utilisés

  • tinyFaceDetector - Détection rapide
  • faceLandmarkNet - Landmarks extraction
  • faceRecognitionNet - Face descriptor (128D vector)
