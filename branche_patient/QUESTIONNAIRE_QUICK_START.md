# Questionnaire System - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### 1️⃣ Update Database (1 min)
```bash
npx prisma generate
npx prisma db push
```

### 2️⃣ Create Doctor Dashboard Page (1 min)
File: `/app/dashboard/questionnaires/page.tsx`
```tsx
import QuestionnaireManagement from "@/components/QuestionnaireManagement";
import { redirect } from "next/navigation";
import { getSession } from "@/lib/actions/auth.actions";

export default async function Page() {
  const session = await getSession();
  if (!session || session.user.role !== "DOCTOR") redirect("/login");
  return <QuestionnaireManagement />;
}
```

### 3️⃣ Create Patient Response Page (1 min)
File: `/app/patient/questionnaires/[id]/page.tsx`
```tsx
import PatientQuestionnaire from "@/components/PatientQuestionnaire";

export default function Page({ params }) {
  return <PatientQuestionnaire assignmentId={params.id} />;
}
```

### 4️⃣ Add Navigation Link (1 min)
Add to your dashboard navigation:
```tsx
<Link href="/dashboard/questionnaires">Questionnaires</Link>
```

### 5️⃣ Test It! (1 min)
- Go to `/dashboard/questionnaires`
- Click "Create Questionnaire"
- Select specialty & generate with AI
- Create → Assign → Patient completes → View responses

## 🎯 Use Case Examples

### Example 1: Cardiology Follow-up
**Doctor:**
1. Create → AI Generate → Cardiology → 5 Questions Auto-generated
2. Assign to 10 patients with 7-day deadline
3. Track completion in real-time

**Patient:**
1. See "Cardiology Follow-up" questionnaire
2. Answer 5 questions about chest pain, shortness of breath, etc.
3. Submit → Done

**Doctor:**
1. View all 10 responses
2. Export for medical records
3. Track who completed vs. pending

**Doctor:**
1. Create → Manual Create → 8 custom questions

3. Assign with 14-day deadline

**Patient:**
1. Fill out surgical recovery questions
2. Rate pain levels using 1-10 scale
3. Submit

**Doctor:**
1. Monitor recovery using responses
2. Identify patients needing follow-up

## 🚨 Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| "Cannot find module" | Run `npm install`, then `npx prisma generate` |
| "Only doctors can create" | Ensure user.role = "DOCTOR" |
| Database error | Run `npx prisma db push` |
| Components missing | Install Shadcn/UI components: `npx shadcn-ui@latest add card` |
| API returns 401 | Check authentication session |

## ✨ Key Features

### 1. Question Types
```
TEXT          - Text input: "What's your main concern?"
TEXTAREA      - Long text: "Describe symptoms in detail"
NUMBER        - Numeric: "Your weight (kg)"
RATING        - 1-10 scale: "Pain level?"
YESNO         - Yes/No: "Do you have fever?"
```

- **PENDING** - Not started
- **IN_PROGRESS** - Partially answered
- **COMPLETED** - All questions answered
- **EXPIRED** - Due date passed

- Text answers
- Numeric values
- Complex JSON data
- Multiple selections
