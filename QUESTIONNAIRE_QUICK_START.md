# Questionnaire System - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### 1️⃣ Update Database (1 min)

```bash
npx prisma generate
npx prisma db push
```

### 2️⃣ Create Doctor Dashboard Page (1 min)

File: `/app/dashboard/questionnaires/page.tsx`

```tsx
import QuestionnaireManagement from "@/components/QuestionnaireManagement";
import { redirect } from "next/navigation";
import { getSession } from "@/lib/actions/auth.actions";

export default async function Page() {
  const session = await getSession();
  if (!session || session.user.role !== "DOCTOR") redirect("/login");
  return <QuestionnaireManagement />;
}
```

### 3️⃣ Create Patient Response Page (1 min)

File: `/app/patient/questionnaires/[id]/page.tsx`

```tsx
import PatientQuestionnaire from "@/components/PatientQuestionnaire";

export default function Page({ params }) {
  return <PatientQuestionnaire assignmentId={params.id} />;
}
```

### 4️⃣ Add Navigation Link (1 min)

Add to your dashboard navigation:

```tsx
<Link href="/dashboard/questionnaires">Questionnaires</Link>
```

### 5️⃣ Test It! (1 min)

- Go to `/dashboard/questionnaires`
- Click "Create Questionnaire"
- Select specialty & generate with AI
- Create → Assign → Patient completes → View responses

## 📋 What You Get

| Feature                   | Details                                     |
| ------------------------- | ------------------------------------------- |
| **Create Questionnaires** | Manual or AI-generated by specialty         |
| **Assign Patients**       | Multiple patients, set due dates            |
| **9 Question Types**      | Text, Rating, Yes/No, Multiple Choice, etc. |
| **Patient Responses**     | Progress tracking, validation, drafts       |
| **Doctor Dashboard**      | View all responses, export, status tracking |
| **AI Generation**         | 16 medical specialties pre-configured       |
| **Role Protection**       | Doctors only, data isolation                |

## 🎯 Use Case Examples

### Example 1: Cardiology Follow-up

**Doctor:**

1. Create → AI Generate → Cardiology → 5 Questions Auto-generated
2. Assign to 10 patients with 7-day deadline
3. Track completion in real-time

**Patient:**

1. See "Cardiology Follow-up" questionnaire
2. Answer 5 questions about chest pain, shortness of breath, etc.
3. Submit → Done

**Doctor:**

1. View all 10 responses
2. Export for medical records
3. Track who completed vs. pending

### Example 2: Post-Surgery Assessment

**Doctor:**

1. Create → Manual Create → 8 custom questions
2. Duplicate for all post-op patients
3. Assign with 14-day deadline

**Patient:**

1. Fill out surgical recovery questions
2. Rate pain levels using 1-10 scale
3. Submit

**Doctor:**

1. Monitor recovery using responses
2. Identify patients needing follow-up

## 🤖 AI Features

### Built-in Specialties (16 types)

- Cardiology → Heart/BP/Chest questions
- Neurology → Headache/Memory questions
- Orthopedics → Joint/Pain questions
- Pulmonology → Breathing/Cough questions
- Psychiatry → Mental health questions
- Gastroenterology, Endocrinology, Rheumatology, Nephrology, Hematology, Oncology, Dermatology, ENT, Ophthalmology, General Medicine, Other

### Generate Options

**Automatic:** Questions auto-populate based on specialty
**Custom:** Add specific instructions for personalized questions

### Future AI Upgrade

Replace the template-based generation in `/api/questionnaires/generate-ai/route.ts` with:

```typescript
// OpenAI
const response = await openai.chat.completions.create({...})

// Or Anthropic Claude
const message = await client.messages.create({...})
```

## 🔐 Security Built-in

✅ Session authentication required
✅ Doctors only see their templates
✅ Patients only see their assignments
✅ Role-based access control
✅ Server-side validation
✅ Cascading deletes prevent orphaned data

## 📊 Data Schema Overview

```
Doctor creates Template
        ↓
  Doctor assigns to Patient
        ↓
   Patient responds
        ↓
  Doctor reviews responses
```
